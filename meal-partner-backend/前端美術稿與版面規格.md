## 3. 版面架構（App Shell）

```
body（#e8e4dc 外層）
└── .app[data-view]（max-width 依斷點變化，背景 --bg-cream）
    ├── header.header
    │   ├── .header-top（頭像 28×28 + 標題）
    │   └── .header-banner--home（首頁專用橫幅，非首頁隱藏）
    ├── main.view × N（一次只顯示一個，[hidden] 強制 display:none）
    ├── button.fab（僅 data-view="home" 顯示）
    └── nav.bottom-nav（固定底部 / 桌面左側）
```

### 3.1 視圖切換

- `app.js` 透過 `switchView(viewKey)` 切換頁面，並設定 `.app` 的 `data-view`。
- `[hidden]` 與 `.view[hidden]` 使用 `display: none !important`，避免其他 CSS 覆蓋。
- 飯局詳情、已加入、評價、新增飯局視為 **home 延伸流程**，底部導覽仍高亮 `home`。

### 3.2 頁面一覽

| viewKey | HTML id | 說明 |
|---------|---------|------|
| `login` | `view-login` | 登入（未登入點個人帳號時） |
| `home` | `view-home` | 首頁：搜尋、篩選、我的/其他飯局 |
| `notifications` | `view-notifications` | 通知中心 |
| `chat` | `view-chat` | 聊天室列表 + 對話面板 |
| `partyDetail` | `view-party-detail` | 飯局詳情 |
| `partyJoined` | `view-party-joined` | 已加入成員頁 |
| `rating` | `view-rating` | 評價成員 |
| `create` | `view-create` | 新增飯局表單 |
| `profile` | `view-profile` | 個人帳號（可編輯） |

## 4. 響應式斷點

| 斷點 | 行為摘要 |
|------|----------|
| 預設（&lt; 360px） | 手機極小屏：縮圖 60px、篩選文字截斷 |
| &lt; 420px | 飯局卡片固定 `height: 178px`；操作按鈕直排 |
| 360px–767px | **手機版主體**：底部導覽、FAB、單欄內容 |
| 480px–767px | 可選「手機框」：`body` 置中，`.app` 圓角 24px + 陰影 |
| ≥ 768px | **桌面版**：左側固定導覽 92px、內容 max 1180px、首頁雙欄 grid |
| 768px–920px | 登入改單欄；飯局詳情改單欄圖文 |
| 768px–980px | 個人帳號雙欄改單欄 |

### 4.1 手機版（&lt; 768px）

| 區塊 | 規格 |
|------|------|
| `.app` | `max-width: 430px`，置中，`min-height: 100dvh` |
| `.view` padding | 左右 16px；底部留空 = 導覽 + FAB + safe-area |
| `.bottom-nav` | 固定底部，全寬 max 430px，高 72px + safe-bottom，上圓角 16px |
| `.fab` | 固定右下，圓形 56px，藍色 `#5a8fa8` |
| `.header-banner` | 高 92px，不對稱圓角 `4px 48px 48px 4px` |
| 聊天對話 | `.chat-room-panel` 全螢幕 overlay（fixed inset） |

### 4.2 桌面版（≥ 768px）

| 區塊 | 規格 |
|------|------|
| `.app` | `max-width: 1180px`，`padding: 28px 36px 48px 144px`（左留給導覽） |
| `.bottom-nav` | `position: fixed`，左側垂直，`width: 92px`，`border-radius: 28px` |
| `.fab` | 右下 `40px`，依視窗與 1180px 置中計算 |
| `#view-home` | `display: grid`，`grid-template-columns: repeat(2, 1fr)`；toolbar 跨兩欄 |
| 詳情/建立/個人/評價卡片 | `max-width: 760px`（個人帳號 editable 可至 980px） |
| 通知 / 聊天列表 | 雙欄 grid |
| 聊天 | 左列表 360px + 右對話區；未選聊天室時右側顯示提示 pseudo 文字 |

## 10. 後端 / 新前端實作建議

1. **色彩與圓角**：優先沿用 §2 tokens，確保 App 與 Prototype 一致。  
2. **卡片比例**：飯局列表建議維持 **2.35:1** 橫向卡片，避免狀態標籤撐高版面。  
3. **狀態色**：前後端共用同一套狀態 enum → CSS class 映射（§2.2）。  
4. **桌面導覽**：≥768px 改左側直欄，內容區左 padding 约 **144px**。  
5. **安全區**：行動端保留 `safe-area-inset` 於底部導覽與 FAB。  
6. **刪除 UX**：通知 / 聊天室用右上角 ×；已取消飯局刪除僅在詳情頁文字按鈕。
